播放设置

播放时先尝试默认音质，若该音质不可用则自动降级尝试更低音质，直至最低音质都失败才反馈播放失败。

外部搜索接口

供外部系统调用的标准化搜索接口，支持通过关键词搜索歌曲并返回播放地址。

POST
请求 / 响应示例 
{
  // 请求体
  "keyword": "歌曲名称",
  "hint": {
    "title": "歌曲名称",
    "artist": "歌手名",
    "duration": 240
  },
  "quality": "320k"
}

// 成功响应
{
  "code": 0,
  "msg": "success",
  "data": {
    "title": "演员",
    "artist": "薛之谦",
    "album": "几个薛之谦",
    "duration": 240,
    "cover_url": "https://example.com/cover.jpg",
    "url": "https://example.com/song.mp3",
    "source_data": {
      "platform": "netease",
      "quality": "320k",
      "songInfo": { "..." }
    }
  }
}

// 错误码: 0=成功 400=参数错误 404=未找到 500=服务器错误
